HADES STOCK PRO - Android APK 제작방법

1. Android Studio 설치
2. 이 ZIP 압축 해제
3. Android Studio 실행
4. Open 클릭
5. 압축 해제한 폴더 선택
6. Gradle Sync가 끝날 때까지 대기
7. 상단 메뉴 Build 클릭
8. Build Bundle(s) / APK(s) 클릭
9. Build APK(s) 클릭
10. 완료 후 app/build/outputs/apk/debug/app-debug.apk 확인
11. 휴대폰으로 APK 복사 후 설치

주의:
- 처음 설치 시 Android가 '알 수 없는 앱 설치' 허용을 물어볼 수 있습니다.
- 개인용 테스트 목적이면 debug APK로 충분합니다.
- Google Play 등록용이면 release signing이 필요합니다.
