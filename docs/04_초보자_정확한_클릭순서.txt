HADES STOCK PRO APK 만들기 - 초보자 클릭 순서

1. ZIP 압축 해제
2. Android Studio 실행
3. Open 클릭
4. 압축 해제한 폴더 hades_android_v16 선택
5. 오른쪽 아래 또는 상단에서 Gradle Sync가 끝날 때까지 대기
6. 상단 메뉴 Build 클릭
7. Build Bundle(s) / APK(s) 클릭
8. Build APK(s) 클릭
9. 완료 알림에서 locate 클릭
10. app-debug.apk를 휴대폰으로 복사
11. 휴대폰에서 APK 눌러 설치

오류가 나면:
- File > Sync Project with Gradle Files 실행
- Build > Clean Project 실행
- 다시 Build APK(s) 실행
