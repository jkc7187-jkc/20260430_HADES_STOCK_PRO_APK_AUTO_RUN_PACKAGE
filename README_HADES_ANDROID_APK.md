# HADES STOCK PRO Android APK V1

개인 전용 투자 분석 앱을 Android APK로 빌드하기 위한 Android Studio 프로젝트입니다.

## 핵심
- BAT/Node/npm 없이 Android WebView에서 실행
- `app/src/main/assets/index.html` 내장
- 추천종목, 이동평균선 차트, 공시 메모, 수익형 프리미엄 구조 포함

## APK 생성
Android Studio에서 이 폴더를 열고:

`Build > Build Bundle(s) / APK(s) > Build APK(s)`

생성 위치:

`app/build/outputs/apk/debug/app-debug.apk`
